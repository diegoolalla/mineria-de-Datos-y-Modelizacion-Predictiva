# Machine-learning-practica-1
Tarea machine learning practica 1


Easy Loans es una empresa financiera que concede préstamos para compras en distintos comercios y, en este caso práctico, tu objetivo es analizar las operaciones del ejercicio 2023 en Tableau para obtener insights que ayuden a tomar decisiones estratégicas, identificando patrones de comportamiento, evaluando la calidad de los préstamos y detectando riesgos, anomalías y oportunidades de mejora ligadas tanto a la modernización del análisis como a la eficiencia y rentabilidad del negocio. Para ello, debes conectar Tableau Desktop al archivo Easy Loans Operaciones 2023.xlsx, revisar calidad e integridad (tipos de datos, nulos e inconsistencias), modelar correctamente las relaciones entre las tablas Orders (order_id, created_at, status, amount, merchant_id, country), Refunds (order_id, refunded_at, amount) y Merchant (merchant_id, name), filtrar las operaciones para quedarte solo con tiendas europeas excluyendo Marruecos, y generar una extracción guardando el proyecto en formato .twbx para compartirlo sin problemas. En la parte analítica, se te pide crear cálculos clave como el Promedio del importe, el Total Comercios, el Máximo y Mínimo del préstamo, el Valor Acumulado con RUNNING SUM, el Promedio Total con una LOD FIXED y el Total Reembolsos, además de construir un parámetro interactivo llamado “Valor Préstamo Mínimo” que permita filtrar dinámicamente las operaciones mediante un cálculo del tipo Amount > [Valor Préstamo Mínimo] aplicado como filtro True en todas las visualizaciones (mostrando el parámetro en el dashboard, no el filtro). Como entregables, debes crear una tabla de KPIs con Valor Acumulado, Promedio, Promedio Total, Máximo, Mínimo, Total Comercios y Total Devoluciones, un mapa por país coloreado por Promedio, un gráfico de áreas con el acumulado diario por país, y una vista de desviación para ver qué operaciones están por encima o por debajo del promedio, además de incorporar filtros globales de País, rango de fechas y el criterio del préstamo mínimo, integrarlo todo en un dashboard final claro y usable, y añadir una acción para que al hacer clic en el mapa se filtren el resto de visualizaciones. De forma opcional, puedes mejorar la experiencia de usuario con un diseño personalizado, navegación más intuitiva, contenedores ocultables, iconos o selectores de métricas, y como bonus puedes publicar el dashboard en Tableau Cloud (compartiéndolo solo con el profesor) o en Tableau Public, aportando captura y enlace; finalmente, la entrega obligatoria es el archivo .twbx por el Campus Virtual, asegurándote antes de que el dashboard funciona correctamente, que los filtros y acciones responden bien y que el archivo se abre sin fallos en otro equipo.
